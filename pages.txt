PROJECT OVERVIEW:
ACAConnect is a comprehensive event management system for academic institutions
with role-based access control, ML-powered event recommendations, FSM workflow,
budget management, and AI chatbot integration.

================================================================================
                              STARTUP SEQUENCE
================================================================================

1. Start MongoDB (ensure running)
2. Start Backend: cd backend && npm start (port 5000)
3. Start Frontend: cd frontend && npm start (port 3000)
4. Start ML Service: cd ml-service && python app.py (port 5001)
5. Start Chatbot Service: cd chatbot-service && python app.py (port 5002)

OR use START_ALL.bat to start all services at once
