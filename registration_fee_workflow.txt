WORKFLOW IMPLEMENTATION:
=======================

NEW REGISTRATION FEE WORKFLOW:

1. EVENT TEAM CREATES EVENT
   - Checks "Registration Fee Required" if fee needed
   - Submits event for approval
   - Cannot set fee amount (Treasurer's responsibility)

2. TREASURER REVIEWS EVENT
   - Sees "Registration Fee Required: Yes/No"
   - If required, sets registration fee amount (₹)
   - Approves event with fee amount
   - Fee gets saved to event document

3. EVENT GETS PUBLISHED
   - Registration fee is now available for participant registration
   - Payment system uses this fee amount
   - Participants see correct fee during registration
