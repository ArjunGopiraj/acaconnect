SYSTEM ARCHITECTURE:
--------------------

Frontend (React) → Backend (Node.js) → ML Service (Flask)
    Port 3000         Port 5000           Port 5001

Request Flow:
1. User selects interests in UI
2. Frontend calls: POST /ml/recommend
3. Backend calls: POST http://localhost:5001/recommend-knn
4. ML service runs Hybrid KNN algorithm
5. Returns recommendations with scores
6. Backend formats and sends to frontend
7. Frontend displays ranked results

If ML service is down:
- Backend catches error
- Falls back to simple tag matching
- Returns 50% scores for matching events
